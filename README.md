# L-System Fractal Architect (Python)

A Tkinter + Turtle based GUI program that generates L-system fractals.

## Features
- GUI input dashboard (Axiom, Rules, Angle, Iterations, Step size)
- Draws inside a Tkinter Canvas using RawTurtle
- Supports L-system commands: F, +, -, [ , ]
- Clear button
- Presets: Koch, Tree, Pentagon
- Fast rendering using tracer(0,0) + update()

## How to Run
1. Install Python 3
2. Run:

```bash
python lsystem_gui.py
